## 💁 웹 서비스 소개

**MINT** 는 콘서트 티켓을 NFT로 발급해주는 서비스입니다.

콘서트를 예매하면 해당 콘서트를 입장할 수 있는 티켓을 NFT로 발급해줍니다.

콘서트가 시작된 후 NFT티켓에 담겨있는 콘텐츠를 확인해 볼 수 있습니다. 원한다면 판매도 진행할 수 있는 거래 플랫폼도 이용할 수 있습니다.

<br />

> SSAFY Wallet이 있다면 누구든 사용가능합니다.
